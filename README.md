# FDF
Fils De Fer - A graphic project in which 3D objects are drawn using isometry.

## Keys for controll
- i: canonic
- p: paralell
- q, w, e, a, s, d - rotate
- z: up Z
- x: down Z
- +: zoom+
- -: zoom-
- m: change color
- f: freak mode on/off
- esc: close programm


## Usage
- git clone 
- cd FDF/fdf
- run make
- ./fdf maps/<mapName>
